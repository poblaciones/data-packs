# data-packs
Conjuntos de cartografías para inicializar Poblaciones

# Introducción

Los data-packs proveen de cartografías de base (geografías) y regiones de recorte con los cuales trabajar datos a nivel nacional.

Actualmente se encuentra disponible el data-pack para Argentina.

Si está utilizando la aplicación en otros países y quiere publicar sus geografías y regiones de recorte como datapacks, escribir a contacto@poblaciones.org.
